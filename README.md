# Sprinkl-o-mat

An automatic irrigation system using a Raspberry Pi Model B (Revision 2.0) 

## Used parts
- Raspberry Pi Model B (Revision 2.0)
- 4-Relay Module 5V
- Capacitive Soil Moisture Sensor v1.2
